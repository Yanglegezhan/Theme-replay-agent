# 情绪周期理论
情绪周期分为五个阶段：
1. **启动期**：首板数量激增，出现破局个股（一字板或秒板），分时图呈45度攻击角，K线连续2-3天放量上涨
2. **高潮期**：涨停家数>10只，全市场炸板率低（<20%），跟风股纷纷涨停，K线连续大阳线，成交额创新高
3. **分化期**：仅前排1-2只龙头涨停，中后排回撤>5%，全市场炸板率飙升（>30%），K线出现长上影线或十字星
4. **修复期**：经历分歧后核心中军或龙头反包，板块指数收复昨日失地，K线出现反包阳线，成交额温和放大
5. **退潮期**：龙头跌停或断板大跌（跌幅>5%），板块整体涨停数骤减（相比前一日减少>50%），K线连续阴线

# K线分析方法（重点）
基于最近14天的K线数据进行周期判断：

## 价格走势特征
- **启动期信号**：
  - 连续2-3天放量上涨，涨幅逐日扩大（如1%→2%→3%）
  - 成交额明显放大（相比前期增长50%以上）
  - K线形态：小阳线→中阳线→大阳线
  
- **高潮期信号**：
  - 连续大阳线（涨幅>3%），可能出现涨停
  - 成交额创近期新高（达到前期2-3倍）
  - K线形态：连续大阳线，实体饱满

- **分化期信号**：
  - 出现长上影线（冲高回落）或十字星（多空分歧）
  - 成交额开始萎缩（相比高点下降30%以上）
  - K线形态：上影线、十字星、小阴小阳交替

- **修复期信号**：
  - 经历回调后出现反包阳线（收复前一日失地）
  - 成交额温和放大（不是暴量）
  - K线形态：先阴后阳，阳线实体大于阴线

- **退潮期信号**：
  - 连续阴线或大幅回调（单日跌幅>3%）
  - 成交额持续萎缩（跌破前期平均水平）
  - K线形态：连续阴线，可能出现跌停

## 量价关系分析
- **价涨量增**：健康上涨，资金积极参与（启动期/高潮期）
- **价涨量缩**：上涨乏力，资金观望（分化期前兆）
- **价跌量增**：恐慌性抛售，情绪崩溃（退潮期）
- **价跌量缩**：下跌动能衰竭，可能见底（修复期前兆）

## 趋势判断
- **短期趋势（3天）**：观察最近3天的连续性
- **中期趋势（7天）**：观察最近7天的累计涨跌幅
- **波动率**：高波动说明情绪剧烈，低波动说明平稳

## 关键K线形态
- **启动信号**：连续小阳线→中阳线，成交量逐步放大
- **高潮信号**：连续大阳线，成交量暴增
- **分化信号**：长上影线、十字星、阴阳交替
- **修复信号**：反包阳线（低开高走收复失地）
- **退潮信号**：连续阴线、大阴线、跌停

# 历史数据分析方法
基于最近7天的历史数据进行周期判断：
- **日K线分析**：观察价格走势、波动率变化，判断资金进出节奏
- **涨停数变化**：分析每日涨停家数的趋势，识别情绪变化拐点
- **趋势一致性**：价格趋势与涨停数趋势的相互验证
- **周期转换信号**：关键指标的组合变化预示周期转换

# 板块强度分类标准
根据板块强度数值判断板块热度：
- **高强度板块**：强度值 ≥ 8000（市场高度关注，资金集中流入）
- **中强度板块**：强度值 2000-8000（市场适度关注，有一定资金参与）
- **低强度板块**：强度值 < 2000（市场关注度低，资金参与有限）

强度值越高，说明该板块越受市场关注，资金参与度越高，情绪周期判断时需要结合强度等级综合分析。

# 风险等级与机会等级判断标准

## 风险等级（risk_level）
根据以下因素综合判断：
- **High（高风险）**：
  - 处于退潮期或分化期后期
  - 炸板率 > 40%
  - 龙头股跌停或大幅下跌
  - 板块强度连续下降超过3天
  - 涨停数相比前一日减少超过50%

- **Medium（中等风险）**：
  - 处于分化期初期或修复期
  - 炸板率在20%-40%之间
  - 龙头股断板但未跌停
  - 板块强度波动但无明显趋势

- **Low（低风险）**：
  - 处于启动期或高潮期
  - 炸板率 < 20%
  - 龙头股连续涨停，梯队完整
  - 板块强度持续上升

## 机会等级（opportunity_level）
根据以下因素综合判断：
- **High（高机会）**：
  - 处于启动期或高潮期初期
  - 板块强度快速上升（单日增幅>20%）
  - 连板高度提升，梯队完整（无断层）
  - 龙头股封单坚决，资金抢筹明显
  - 有持续性催化剂支持

- **Medium（中等机会）**：
  - 处于修复期或高潮期后期
  - 板块强度稳中有升
  - 龙头股表现尚可但跟风不足
  - 需要精选个股，不能盲目追涨

- **Low（低机会）**：
  - 处于分化期或退潮期
  - 板块强度下降
  - 仅前排个别股有机会，操作难度大
  - 建议观望或减仓

# 关键指标分析要求

对于每个关键指标，不仅要列出指标名称和数值，还要提供：
1. **指标解读**：该数值代表什么含义（如"涨停8只，说明板块热度较高，资金参与积极"）
2. **趋势分析**：与前几日对比是上升还是下降（如"较昨日增加3只，呈上升趋势"）
3. **影响判断**：该指标对周期阶段的支撑作用（如"连板高度提升至5板，支撑板块处于高潮期判断"）

示例格式：
- 涨停家数：8只（较昨日+3只，呈上升趋势，资金参与积极，支撑高潮期判断）
- 最高连板：5连板（较昨日+1板，空间打开，龙头效应明显）
- 炸板率：15%（较低，说明封板坚决，情绪良好）
